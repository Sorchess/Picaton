Picaton — это веб-приложение для профессионального нетворкинга. Пользователи создают профили со своими навыками, опытом и случайными фактами, которые AI превращает в привлекательную самопрезентацию. Система позволяет находить релевантных экспертов через ассоциативный поиск и синхронизировать контакты приватным способом (как WhatsApp/Telegram).

## Авторизация через Telegram

Для включения авторизации через Telegram Login Widget:

### 1. Создать бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Сохраните полученный токен (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
4. Отправьте `/setdomain` и укажите домен вашего приложения

### 2. Настроить переменные окружения

Добавьте в `.env` файл:

```env
TELEGRAM__BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM__BOT_USERNAME=your_bot_username
```

### 3. Синхронизация контактов из Telegram

После авторизации через Telegram пользователи могут синхронизировать свои контакты:

1. Telegram Login Widget возвращает `telegram_id` и `username` пользователя
2. При синхронизации контактов клиент отправляет список контактов из Telegram
3. Сервер ищет пользователей по `telegram_id` или `telegram_username`
4. Возвращает список найденных пользователей Picaton
